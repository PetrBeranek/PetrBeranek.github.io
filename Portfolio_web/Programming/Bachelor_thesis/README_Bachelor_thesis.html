Bachelor thesis

Finished in May 2015
My bachelor thesis was written in Czech language.

Abstract:
The goal of this thesis is to analyze 3D reconstruction application ArchiRec3D and to design and implement several tools, which give user new possibilities of plane editing. One of the tools allows user to create/modify borders, which are used to trim the planes. Other tool allows user to apply texture to these trimmed planes. The next important tool allows user to create 3D objects form trimmed plane. Thesis contains analysis of the most important parts of the application. There are also chapters containing problem analysis, design, implementation and testing of these tools. The result is the modified application which allows user to create more detailed textured objects, which was not possible before.

About ArchiRec3D
ArchiRec3D(AR) is an experimental program for 3D reconstruction created by David Sedláček implemented in Java using Java3D API. AR uses calibrated scene created by programs such as Bundler [1]. Calibrated scene is created from photographies of object and consists of point cloud, cameras, photographies and informations about connections between photographies and points of point cloud (especially position of point in photography). AR enables visualisation of the scene and gives user several tools to work with scene. User is able to select parts of photographies (walls, doors, roof) by drawing and plane is created in scene. The position and direction of plane is determined by points in selected part of photography. RANSAC algorithm is used to create best approximation. AR is designed in such way, that user have to do minimum work. User draws simple scribbles, program does the most of the work. User is able to refine the plane by adding scribbles to different photographies. AR allows user to create any amount of planes and user can define which plane should clip other planes. Unfortunately user was unable to define shape of planes. Basic plane was always rectangular with size defined by properties of points used to create it.

Note:
The most important part of my work was to orientate in ArchiRec3D. Program had no documentation and only few parts of code were commented (sometimes even name of methods didn't say much about what it does).

My task:
Create tool that allows user to create border of any plane.
Create tool that will enable user to refine border of any plane in any photography in the scene.
Create tool to apply texture to the plane. (Texture is created from photography chosen by user.)
Create tool for extruding plane to create 3D object.
Create import module for scenes created by VisualSFM [2].

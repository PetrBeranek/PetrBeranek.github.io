Beetle

Finished in December 2013

Reason to create project:

This project was created for course MGA [1] during my bachelor studies.
Task was to create model of any car we wanted. I choose to create beetle, because of its round shape.

About project:

Project was created in Blender, renders were made using Blender renderer.

This was the first 3D model I have ever created. It is not perfect and now I would make it differently and more detailed, but I am still proud of it.
Scene contains, as name suggests, Volkswagen Beetle.

[1] MGA (Multimediální a grafické aplikace - Multimedia and graphic applications)

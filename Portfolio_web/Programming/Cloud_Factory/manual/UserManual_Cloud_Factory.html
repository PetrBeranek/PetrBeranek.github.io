Controls:

Movement:
  • Mouse - hold any mouse button to rotate
  • F - activate object at mouse position
  • W - move forward
  • S - move backward
  • A - move left
  • D - move right
  • Space - move up
  • C - move down

Others:
  • O - increase fog density by 0.1
  • L - decrease fog density by 0.1
  • P - reset fog density to 0 (turn off)
  • R - load and set parameters from config.txt (set lights and fog)

Special:
  • F1 - preset camera position 1
  • F2 - preset camera position 2
  • F3 - preset camera position 3
  • F4 - attach camera to butterfly
  • F5 - attach camera to curve

  • F9 - basic shader
  • F10 - shader experiment (shading by hatching)
  • F11 - shader experiment (quantization)

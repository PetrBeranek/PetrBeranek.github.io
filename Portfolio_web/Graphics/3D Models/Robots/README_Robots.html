Robots

Finished in July 2014

Reason to create project:

These projects I created in free time just for fun. I thought they may be useful one day.

About project:

Project was created in Maya 2014 (Student licence), rendered by Mental Ray.

Mechanoid is humanoid robot with many pistons, cog wheels and other moving parts.
Excavator is non humanoid robot named because of his right hand.
Most mechanical parts are connected using expressions and constraints.

Scene is expected to be used in Maya. Exported .fbx file probably doesn't have all the functions or correct look. (.fbx scene is just for preview or to access models even without Maya)

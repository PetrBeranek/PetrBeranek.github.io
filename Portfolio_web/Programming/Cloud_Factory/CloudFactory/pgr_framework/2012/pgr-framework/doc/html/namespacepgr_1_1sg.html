Detailed Description

Contains all scene graph classes.

Author:
Tomas Barak, Petr Felkel, Jaroslav Sloup
Date:
2011-2013

All nodes are derived from the base class SceneNode. The ResourceManager class takes care of the resource reference counting, as the textures and models are shared between nodes

Arien

Finished in July 2014

Reason to create project:

This project was created in free time. I wanted to recreate one of my wire statues, but this time in more mechanical way.

About project:

Project was created in Maya 2014 (Student licence), rendered by MentalRay. Scene contains Arien, mechanical monster with four arms made of steel plates, pistons and cog wheels.
all pistons and wheels are tied together using expressions and constraints so they seems to work while Arien moves.
During this project I learnt that even thou model looks great in Maya, when exported into other formats they can be highly altered (And that using scale less than 0 instead of mirroring may look good in maya, but it messes up the normals). Another thing learned during creation was that parenting parts to skeleton instead of skinning is faster, but can cause big problems then exported.
Despite these problems (which I will have to solve once I decide to use this model) model came out better than I expected in beginning.
I also managed to keep triangle count as low as I could (30 thousands).

Scene is expected to be used in Maya. Exported .fbx file probably doesn't have all the functions or correct look. (.fbx scene is just for preview or to access models even without Maya)
